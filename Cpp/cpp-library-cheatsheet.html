std::stack（栈）
s.push(val)入栈
s.pop()出栈（不返回值）
s.top()获取栈顶元素
s.size() / s.empty()大小 / 判空

s.insert(val)插入元素（自动去重）
s.erase(val)删除指定元素
s.find(val)查找元素
s.count(val)判断元素是否存在（0 或 1）
s.lower_bound(val)第一个 >= val 的迭代器（仅 set）
s.upper_bound(val)第一个 > val 的迭代器（仅 set）
s.size() / s.empty() / s.clear()大小 / 判空 / 清空

std::make_pair(a, b)创建 pair
